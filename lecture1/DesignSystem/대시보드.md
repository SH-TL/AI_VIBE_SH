## 1. 타이포그래피 시스템

### CSS 변수

```css
:root {
  /* Font Family */
  --font-family-base: 'Inter', 'Noto Sans KR', sans-serif;

  /* Font Size */
  --font-size-kpi:      32px;   /* KPI 대형 수치 (201, 4.890, $30,256.23) */
  --font-size-kpi-lg:   28px;   /* KPI 중형 수치 (25410, 1352) */
  --font-size-h1:       20px;   /* 페이지 타이틀 (Analytics) */
  --font-size-h2:       16px;   /* 섹션 타이틀 (Sales dynamics) */
  --font-size-h3:       14px;   /* 카드 라벨, 사이드바 메뉴 */
  --font-size-body:     13px;   /* 테이블 본문, 서브 텍스트 */
  --font-size-sm:       12px;   /* 테이블 헤더, 캡션, 메타 */
  --font-size-xs:       11px;   /* 변화율 (▲7.5%), 날짜 */

  /* Font Weight */
  --font-weight-bold:     700;
  --font-weight-semibold: 600;
  --font-weight-medium:   500;
  --font-weight-regular:  400;

  /* Line Height */
  --line-height-tight:   1.1;   /* KPI 수치 */
  --line-height-snug:    1.3;   /* 타이틀, 카드 라벨 */
  --line-height-normal:  1.5;   /* 본문 */
  --line-height-relaxed: 1.6;   /* 설명 텍스트 */
}
```

### 타이포그래피 스케일 사용 가이드

| 변수명 | 크기 | 굵기 | 용도 |
|--------|------|------|------|
| `--font-size-kpi` | 32px | 700 | KPI 대형 수치 (Orders, Users) |
| `--font-size-kpi-lg` | 28px | 700 | KPI 중형 수치 (Month total, Revenue) |
| `--font-size-h1` | 20px | 600 | 페이지 타이틀 |
| `--font-size-h2` | 16px | 600 | 섹션/카드 타이틀 |
| `--font-size-h3` | 14px | 500 | 사이드바 메뉴, 카드 라벨 |
| `--font-size-body` | 13px | 400 | 테이블 셀, 서브텍스트 |
| `--font-size-sm` | 12px | 500 | 테이블 헤더, 날짜 |
| `--font-size-xs` | 11px | 400–500 | 변화율, 보조 수치 |

### 반응형 타이포그래피

```css
@media (max-width: 900px) {
  :root {
    --font-size-kpi:    24px;
    --font-size-kpi-lg: 20px;
    --font-size-h1:     18px;
    --font-size-h2:     14px;
  }
}

@media (max-width: 600px) {
  :root {
    --font-size-kpi:    20px;
    --font-size-kpi-lg: 18px;
  }
}
```

## 7. MUI 테마 연동 코드

```js
import { createTheme } from '@mui/material/styles';

const dashboardTheme = (mode = 'light') => createTheme({
  palette: {
    mode,
    background: {
      default: mode === 'light' ? '#F4F6FA' : '#13131F',
      paper:   mode === 'light' ? '#FFFFFF'  : '#1C1C2E',
    },
    text: {
      primary:   mode === 'light' ? '#1A1D2E' : '#F1F5F9',
      secondary: mode === 'light' ? '#6B7280' : '#94A3B8',
    },
    success: { main: '#10B981' },
    error:   { main: '#EF4444' },
  },
  typography: {
    fontFamily: "'Inter', 'Noto Sans KR', sans-serif",
    h1: { fontSize: '1.25rem', fontWeight: 600, lineHeight: 1.3 },
    h2: { fontSize: '1rem',    fontWeight: 600, lineHeight: 1.3 },
    body1: { fontSize: '0.8125rem', fontWeight: 400, lineHeight: 1.5 },
    body2: { fontSize: '0.75rem',   fontWeight: 400, lineHeight: 1.5 },
    caption: { fontSize: '0.6875rem', fontWeight: 400 },
  },
  spacing: 4, /* 기본 단위 4px */
  shape: {
    borderRadius: 10, /* 기본 radius: 10px (--radius-md) */
  },
  breakpoints: {
    values: { xs: 0, sm: 600, md: 900, lg: 1200, xl: 1536 },
  },
  components: {
    MuiCard: {
      styleOverrides: {
        root: {
          borderRadius: 14,
          padding: 20,
          boxShadow: 'none',
          border: '1px solid',
          borderColor: mode === 'light' ? '#E5E7EB' : '#2D2D44',
        },
      },
    },
    MuiTableCell: {
      styleOverrides: {
        root: {
          fontSize: '0.8125rem',
          padding: '12px 16px',
          borderBottom: '1px solid',
          borderColor: mode === 'light' ? '#F3F4F6' : '#252538',
        },
        head: {
          fontSize: '0.75rem',
          fontWeight: 500,
        },
      },
    },
    MuiChip: {
      styleOverrides: {
        root: {
          height: 24,
          fontSize: '0.6875rem',
          fontWeight: 600,
          borderRadius: 999,
        },
      },
    },
  },
});

export default dashboardTheme;
```
