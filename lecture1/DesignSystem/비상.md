## 1. 타이포그래피 시스템

### CSS 변수

```css
:root {
  /* Font Family */
  --font-family-base: 'Noto Sans KR', 'Apple SD Gothic Neo', sans-serif;

  /* Font Size */
  --font-size-hero:    40px;   /* 히어로 대제목 (공부의 힘! 완자 공부력 시리즈) */
  --font-size-display: 32px;   /* 섹션 대제목 */
  --font-size-h1:      28px;   /* 섹션 타이틀 (비상이 추천하는 초등 교재) */
  --font-size-h2:      24px;   /* 서브 섹션 타이틀 */
  --font-size-h3:      18px;   /* 카드 제목, 강조 텍스트 */
  --font-size-body-lg: 16px;   /* 주요 본문 */
  --font-size-body:    14px;   /* 기본 본문, 네비게이션 메뉴 */
  --font-size-sm:      13px;   /* 보조 텍스트, 메타 정보 */
  --font-size-xs:      12px;   /* 캡션, 날짜, 카테고리 */
  --font-size-badge:   11px;   /* 뱃지, 태그 */

  /* Font Weight */
  --font-weight-bold:    700;
  --font-weight-semibold: 600;
  --font-weight-medium:  500;
  --font-weight-regular: 400;

  /* Line Height */
  --line-height-tight:  1.2;   /* 히어로, 대제목 */
  --line-height-snug:   1.35;  /* 섹션 타이틀 */
  --line-height-normal: 1.5;   /* 카드 제목, 소제목 */
  --line-height-relaxed: 1.6;  /* 본문 텍스트 */
  --line-height-loose:  1.75;  /* 리뷰, 긴 설명 텍스트 */
}
```

### 타이포그래피 스케일 사용 가이드

| 변수명 | 크기 | 굵기 | 용도 |
|--------|------|------|------|
| `--font-size-hero` | 40px | 700 | 히어로 배너 메인 문구 |
| `--font-size-display` | 32px | 700 | 페이지 대표 섹션 제목 |
| `--font-size-h1` | 28px | 700 | 섹션 타이틀 |
| `--font-size-h2` | 24px | 700 | 서브 섹션 타이틀 |
| `--font-size-h3` | 18px | 600 | 카드 제목, 강조 항목 |
| `--font-size-body-lg` | 16px | 400 | 주요 본문, 카드 설명 |
| `--font-size-body` | 14px | 400–500 | 기본 본문, GNB 메뉴 |
| `--font-size-sm` | 13px | 400 | 보조 텍스트, 메타 |
| `--font-size-xs` | 12px | 400 | 날짜, 카테고리, 캡션 |
| `--font-size-badge` | 11px | 600 | 뱃지, 태그 라벨 |

### 반응형 타이포그래피

```css
/* 모바일 (~600px) */
@media (max-width: 600px) {
  :root {
    --font-size-hero:    26px;
    --font-size-display: 22px;
    --font-size-h1:      20px;
    --font-size-h2:      18px;
    --font-size-h3:      16px;
  }
}

/* 태블릿 (600px~900px) */
@media (min-width: 600px) and (max-width: 900px) {
  :root {
    --font-size-hero:    32px;
    --font-size-display: 26px;
    --font-size-h1:      24px;
  }
}
```
