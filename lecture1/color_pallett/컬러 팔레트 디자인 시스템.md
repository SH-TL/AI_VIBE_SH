## 컬러 사용 가이드

### Primary — `#EAB500` (황금 옐로우)
- 페이지에서 가장 먼저 시선을 끌어야 하는 요소에만 사용
- 메인 헤드라인 (H1, H2) 텍스트 색상
- 주요 CTA 버튼 배경색 — 텍스트는 반드시 `#180805` (다크) 사용
- 섹션 구분 강조선 (border-bottom, underline 효과)
- 네비게이션 활성 탭 인디케이터

### Primary Light — `#FFD000`
- 버튼 hover 상태 → 배경을 밝혀 피드백 제공
- 텍스트 링크 hover 밑줄 색상
- 아이콘 active 상태 색상

### Primary Dark — `#C49200`
- 버튼 active(눌림) 상태
- 카드 외곽선, Divider 라인
- 그런지 텍스처 느낌의 배지/태그 테두리

### Secondary — `#6070C8` (블루-퍼플)
- 인물 사진, 이미지에 CSS `mix-blend-mode` 또는 `filter`로 듀오톤 적용
- 보조 버튼(outlined variant) 색상
- 섹션 배경 악센트 (전체 배경과 구분되는 카드 내부)
- 데이터 시각화, 차트의 보조 데이터 색상

### Accent — `#8090E0`
- 링크 hover 색상
- 툴팁, 팝오버 배경
- Secondary 색상 위에 올라가는 텍스트 강조

### Background Primary — `#180805` (다크 버건디 블랙)
- 페이지 전체 기본 배경
- 홀수 섹션 배경

### Background Secondary — `#0D0402`
- 카드, 모달, 드롭다운 내부 배경
- 짝수 섹션 배경 (극히 미묘한 구분)
- 코드 블록 배경

### Text Primary — `#EAB500`
- 모든 헤드라인 (H1~H3)
- 강조 인용구 (blockquote)
- 포인트 수치 (숫자, 통계)

### Text Secondary — `#F0EAD8` (크림화이트)
- 본문 텍스트, 설명 문장
- 서브 헤드라인 (H4~H6)
- 네비게이션 메뉴 아이템 기본 상태

### Text Muted — `#7A6030`
- Placeholder 텍스트
- 메타 정보 (날짜, 카테고리, 태그)
- 비활성(disabled) 상태 텍스트

## 반응형 고려사항

### 이 팔레트는 다크모드 기본 설계

기본(`:root`) 자체가 다크모드이므로 라이트모드 전환 시 별도 처리 필요:

```css
@media (prefers-color-scheme: light) {
  :root {
    --color-bg-primary: #F5F0E8;
    --color-bg-secondary: #EDE8DC;

    --color-text-primary: #1A0805;
    --color-text-secondary: #2C1A08;
    --color-text-muted: #8A7050;

    --color-border: #C49200;
    --color-border-light: #E0D4A0;

    /* Primary/Secondary 인터랙션 컬러는 동일 유지 */
    --color-primary: #C49200;
    --color-primary-light: #EAB500;
    --color-button-primary: #C49200;
    --color-button-hover: #EAB500;
  }
}
```

### 모바일 (~600px) 고려사항
- 버튼 최소 높이 `48px` 확보 (터치 타겟)
- `--color-primary`(`#EAB500`) + `--color-bg-primary`(`#180805`) 대비 비율: **~10:1** → AAA 충족, 작은 텍스트도 안전
- 카드 간 구분이 어두워 흐릿할 수 있으므로 `box-shadow: 0 0 0 1px var(--color-border)` 보완 권장

### 태블릿 (600px~900px)
- 듀오톤 이미지 효과를 `opacity: 0.85`로 낮춰 사진 식별성 확보
- 타이포그래피 크기를 `clamp()`로 유동적으로 처리 권장:
  ```css
  font-size: clamp(1rem, 2.5vw, 1.25rem);
  ```

### 데스크톱 (900px~)
- 그런지 텍스처 배경 이미지 추가 시 `--color-bg-primary` 위에 `mix-blend-mode: multiply` 레이어 권장
- 와이드 스크린에서 `max-width: 1200px`, `margin: auto`로 콘텐츠 폭 제한 필수

### 접근성 (Accessibility)

| 색상 조합 | 대비 비율 | WCAG 기준 |
|-----------|-----------|-----------|
| `#EAB500` on `#180805` | **~10.2:1** | AAA 충족 |
| `#F0EAD8` on `#180805` | **~12.5:1** | AAA 충족 |
| `#FFFFFF` on `#6070C8` | **~4.8:1** | AA 충족 |
| `#7A6030` on `#180805` | **~2.8:1** | AA 미달 — 본문 단독 사용 금지, 보조 정보에만 |

> `--color-text-muted(#7A6030)`는 14px 이하 본문 텍스트에 단독 사용 금지.
> 아이콘 + 텍스트 병행 사용 또는 16px Bold 이상에서만 허용.
