## 컬러 사용 가이드

### Primary (`#2196F3`)
- 주요 CTA 버튼 배경색
- 네비게이션 활성 탭 인디케이터
- 주요 아이콘 및 일러스트 메인 색상
- 헤드라인 강조 텍스트

### Primary Light (`#42B0FF`)
- 버튼 hover/focus 상태
- 카드 위에 올려진 아이콘 광택 효과
- 그라디언트 밝은 끝 지점

### Primary Dark (`#1565C0`)
- 버튼 active(눌린) 상태
- 링크 기본 색상
- 다크 계열 텍스트 강조 (배경이 밝을 때)

### Secondary (`#5BC8FF`)
- 글래스모피즘 카드 테두리 및 내부 하이라이트
- 3D 오브젝트 보조 색상
- 섹션 구분 배지 또는 태그 배경

### Accent (`#00C6FF`)
- 빛 반사, 글로우 효과
- 그라디언트의 시작/끝 포인트
- 특별 강조가 필요한 배지, 알림 점

### Background Primary (`#F0F8FF`)
- 페이지 전체 기본 배경
- 섹션 교차 배경 (홀수 섹션)

### Background Secondary (`#EAF4FF`)
- 카드 내부 배경
- 짝수 섹션 배경 (미묘한 구분)
- 입력 필드 배경

### Text Primary (`#1A2B4A`)
- 모든 헤드라인 (H1~H3)
- 본문 핵심 강조 텍스트

### Text Secondary (`#3B9EF5`)
- 서브 헤드라인, 섹션 타이틀 앞 레이블
- 링크 내 강조 단어

### Text Muted (`#6B7A8D`)
- 본문 설명 텍스트
- Placeholder, 메타 정보 (날짜, 카테고리)
- 비활성 메뉴 항목

### Border (`#D4EAFF`)
- 카드 외곽선
- 구분선 (Divider)

## 반응형 고려사항

### 다크모드 대응

```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary: #0D1B2A;
    --color-bg-secondary: #132337;
    --color-bg-paper: #1A2B3C;

    --color-text-primary: #E8F4FF;
    --color-text-secondary: #5BC8FF;
    --color-text-muted: #8AA4BE;

    --color-border: #2A4060;
    --color-border-light: #1E3050;

    /* Primary/Accent 계열은 다크모드에서도 동일 유지 */
    --color-primary: #2196F3;
    --color-primary-light: #42B0FF;
    --color-button-hover: #64B5F6;
  }
}
```

### 모바일 (~600px) 고려사항
- 버튼 터치 영역: 최소 `44px` 높이 확보 (`--color-button-primary` 적용 시)
- 텍스트 대비: `--color-text-primary` + `--color-bg-primary` 조합은 WCAG AA 기준 충족
- 카드 배경(`--color-bg-secondary`)은 흰 배경에서 구분이 약해지므로 모바일에서 `box-shadow` 보완 권장

### 태블릿 (600px~900px) 고려사항
- 글래스모피즘 카드의 `--color-border` 투명도를 `opacity: 0.6`으로 낮춰 가독성 확보
- 배경 그라디언트 전환을 `--color-bg-secondary` → `--color-bg-primary` 방향으로 부드럽게 처리

### 접근성 (Accessibility)
| 조합 | 대비 비율 | WCAG 기준 |
|------|-----------|-----------|
| `#1A2B4A` on `#F0F8FF` | ~9.5:1 | AAA 충족 |
| `#FFFFFF` on `#2196F3` | ~4.6:1 | AA 충족 |
| `#6B7A8D` on `#FFFFFF` | ~4.2:1 | AA 충족 |
| `#3B9EF5` on `#FFFFFF` | ~3.1:1 | AA Large 충족 (18pt 이상만) |

> `--color-text-secondary(#3B9EF5)`는 작은 본문 텍스트에 단독 사용을 피하고, 18px 이상 또는 Bold 처리된 텍스트에만 사용할 것.

## MUI 테마 연동 코드

```js
import { createTheme } from '@mui/material/styles';

const theme = createTheme({
  palette: {
    primary: {
      main:  '#2196F3',
      light: '#42B0FF',
      dark:  '#1565C0',
    },
    secondary: {
      main: '#5BC8FF',
    },
    background: {
      default: '#F0F8FF',
      paper:   '#FFFFFF',
    },
    text: {
      primary:   '#1A2B4A',
      secondary: '#6B7A8D',
    },
  },
});

export default theme;
```
